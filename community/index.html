Contribution and benchmark routes

Cortrix does not require a CLA. New commits require a DCO 1.1 Signed-off-by line. Read the contribution guide before opening a pull request, use the Cortrix Benchmarks challenge process for published-result disputes, and follow the Code of Conduct.
