Benchmark methodology

Evaluation method first. Numbers only after freeze.

This page documents the intended public benchmark structure before results are frozen. It does not publish performance values, result tables, percentage gains, or claims of superiority.

Framework

Use the BEIR official library and an industry-style BEIR evaluation SOP as the benchmark foundation.

Dataset plan

Use a BEIR five-subset standard loader so test scope is explicit and reproducible.

Metrics

Report NDCG, Recall, MAP, and Precision only after the freeze gate and review process are complete.

Freeze gate

Before public results appear, methodology, datasets, baselines, hardware/software context, and reproducibility notes must be frozen and reviewed.

Not published in this pass

  • No benchmark result values.
  • No result table.
  • No percentage improvement claim.
  • No attack copy against other projects or tools.
  • No claim that a future benchmark guarantees a search, SEM, or GEO outcome.
