# Contributing

Thank you for helping improve cortrix.ai.

We welcome issues and pull requests for:

- Broken links
- Typos
- Documentation clarity
- Accessibility issues
- Layout bugs
- Incorrect or outdated website information with supporting evidence

Please do not use this repository for:

- Private security reports
- Commercial spam
- Advertising or promotional links
- Requests for Cortrix to endorse a third-party service
- Unauthorized brand use
- Product roadmap changes
- Unapproved product claims
- Benchmark, pricing, enterprise, or availability claims that have not been approved by Cortrix maintainers

By submitting an issue, suggestion, or pull request, you grant the Cortrix project maintainers permission to use, edit, reproduce, publish, and display your contribution as part of cortrix.ai and related Cortrix public materials.

Submitting a contribution does not grant you any license to reuse cortrix.ai, website content, Cortrix product messaging, brand assets, names, logos, or brand identifiers.

All pull requests are reviewed by Cortrix maintainers before merge. Maintainers may edit, decline, or close contributions that are outside the scope of this repository.
