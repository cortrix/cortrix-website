Scope / Roadmap

Clear boundaries build trust.

Cortrix uses four public labels so builders can tell what is current open-source work, what is planned, what is blocked, and what remains vision-level thinking.

Current OSSOpen-source semantic storage foundations, public website routes, and agent-readable public documentation.
PlannedWork that has a credible implementation direction but should not be described as already shipped.
BlockedWork that depends on missing design, legal, infrastructure, benchmark, invite, or release decisions.
VisionLonger-term possibilities that should not become current-product promises or CTA copy.

Not current offerings

This page does not promote unannounced hosted, commercial, resilience, certification, or customer-proof commitments as current capabilities.

Launch-day gate

SEO, SEM, and GEO checks are quality and information-architecture constraints, not promises of ranking, conversion, or LLM recommendation.
