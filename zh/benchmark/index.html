性能方法

方法先冻结，数字后发布。

本页只说明公开 benchmark 的结构和冻结门槛，不发布性能值、结果表、提升百分比或优于其它项目的说法。

框架

以 BEIR official library 和行业化 BEIR 评估 SOP 作为性能评估基础。

数据集计划

使用 BEIR 五个子集的标准 loader，让测试范围清晰、可复现。

指标

NDCG、Recall、MAP 和 Precision 只能在 freeze gate 与 review 完成后公开。

冻结门槛

公开数字前，方法、数据集、baseline、软硬件环境和复现说明必须冻结并完成 review。

本轮不发布

  • 不发布 benchmark 结果值。
  • 不发布结果表。
  • 不发布百分比提升。
  • 不写攻击其它项目或工具的文案。
  • 不声称未来 benchmark 会保证搜索、SEM 或 GEO 结果。
