范围 / Roadmap

清楚的边界建立信任。

Cortrix 使用四类公开标签，帮助构建者区分 v1.0 发布目标、已经有方向但未交付的计划、受关键决策阻塞的事项，以及长期愿景。

v1.0 发布目标当前作为 public release 准备的开源语义存储基础；用于文档摄取、检索、记忆、来源追踪、本地工作流和 agent 可读文档。
Coming next已有可信实现方向，但不应描述为已经发布的工作。
Under review依赖设计、法律、基础设施、benchmark、邀请或 release 决策的事项。
长期方向长期可能性，不应进入当前产品承诺或 CTA 文案。

不是当前产品名

本页不把未发布的托管、商业、高可用、认证或客户证明写成当前能力。

发布前门槛

SEO、SEM 和 GEO 是质量与信息架构约束，不等于排名、转化或 LLM 推荐承诺。

未来 /llms/ 主题

公开 /llms/ 主题页仍是未来计划；本轮实现 /llms.txt 和 /docs/llms/，中文镜像为 /zh/docs/llms/。
