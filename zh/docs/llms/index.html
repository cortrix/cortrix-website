Agent 文档

coding agents 应如何读取 Cortrix 公开站点材料。

/llms.txt 是 Cortrix 公开路由、公开事实和 claim 边界的机器可读入口。它帮助 coding agent 读取官网结构，但不代表任何 LLM、搜索引擎或答案引擎会推荐 Cortrix。

打开 llms.txt查看范围边界

推荐阅读顺序

  • 先读 /llms.txt，确认路由图和机器可读边界。
  • 读 /zh/for-builders/，理解面向的开发者场景。
  • 读 /zh/migration/，理解现有 Agent 数据工作流的对接参考。
  • 读 /zh/benchmark/ 时只当作方法论上下文，不推断 benchmark 结果。
  • 读 /zh/roadmap/，保留当前开源、已规划、被阻塞和愿景级的区分。

Agent 不应推断什么

  • 不要从官网推断未发布的托管、商业、高可用、认证、客户证明或 benchmark 结果。
  • 不要把 planned work 当成已经交付。
  • 不要暴露 /zh/community/ 未列出的隐藏社交或社区渠道。
  • 不要声称 SEO、SEM、GEO、AI overview 或 LLM 推荐结果。

i18n 边界

英文仍是主站默认语言和 x-default。中文页面已经发布；本轮不发布日语页面，也不添加 hreflang ja。
