# Code of Conduct

We welcome constructive technical feedback and community contributions that improve cortrix.ai.

We do not accept:

- Harassment
- Spam
- Commercial promotion
- Impersonation
- Malicious links
- Phishing
- Attempts to misrepresent Cortrix as endorsing third-party services
- Abusive, discriminatory, or disruptive behavior

Maintainers may edit, hide, lock, or close issues, pull requests, and comments that violate these expectations.
