What this page is, and what it is not

  • It is a pre-release visual candidate for launch-page review and red-blue evaluation.
  • It is based on BEIR-style retrieval evaluation: Recall@10, NDCG@10, and latency gates.
  • It is not a public measured benchmark result.
  • It is not a claim that Cortrix already beats Qdrant, Milvus, Weaviate, pgvector, Chroma, OpenSearch, Elastic, Vespa, OpenAI, Cohere, or Voyage.
Reference-note for review: public retrieval posts from OpenSearch, Vespa, and Elastic support the baseline categories used in this preview. They are not used as same-run Cortrix comparison results, endorsements, or product rankings.
Release rule: before this page becomes production copy, the projection bars must be replaced by raw JSON, CSV summary, methodology notes, hardware/software details, run count, dataset list, and source snapshots.
