  • Start with /llms.txt for version, route map, source references, and machine-readable boundaries.

What agents must not infer

  • Do not infer unannounced hosted, commercial, resilience, certification, customer-proof, or third-party victory claims from this website.
  • Do not treat source presence or public documentation as proof that a capability is Verified, shipped, production-ready, or security-ready.
  • Use only the official project and community destinations published on /community/.
  • Do not claim SEO, SEM, GEO, AI overview, or LLM recommendation outcomes.
